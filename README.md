# Background Generator

## 📌 Description
This is a simple background generator project built using HTML and CSS.  
It allows users to view and generate different background styles.

## 🚀 Features
- Simple and clean UI
- Background color/style display
- Easy to use

## 🛠️ Technologies Used
- HTML5
- CSS3 (Internal CSS)

## 📂 Project Structure
│── index.html  

## ▶️ How to Run
1. Download or clone this repository  
2. Open index.html in your browser
